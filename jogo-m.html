Instruções

A cada rodada, são abertas 5 cartas com características dos paradigmas da avaliação da aprendizagem.

Relacione cada carta com o paradigma correspondente para marcar pontos!

Você pode abrir novas cartas a qualquer momento. As cartas não respondidas voltam para o banco.

Para ver as cartas já relacionadas, utilize o botão Ver respondidas.

Clique em Sacar cartas para iniciar.

Arraste para os lados para ver as demais cartas.

Clique sobre uma carta para classificá-la.

Arraste para os lados para alternar entre categorias.

Toque sobre o título da categoria para navegar por lista.
